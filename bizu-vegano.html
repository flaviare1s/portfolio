Bizu Vegano Fortaleza

Bizu Vegano é um projeto que mapeia restaurantes e lojas veganas em Fortaleza, ajudando os usuários a encontrar opções de comida vegana na cidade. Originalmente criado com HTML e CSS, o site foi atualizado para utilizar React e Firebase, proporcionando uma interface mais moderna e interativa.

Bizu Vegano is a project that maps vegan restaurants and shops in Fortaleza, helping users find vegan food options in the city. Originally created with HTML and CSS, the site was updated to use React and Firebase, providing a more modern and interactive interface.
